About This Application

This application is similar to Facebook but offers only basic features, such as adding posts.

Technologies Used:
- Frontend: React
- Backend: Node.js
- API Structure: REST

Installation and Setup:
To run the application, install dependencies in both the frontend and backend folders.
After installation, start the application in both directories.

How to Use:
Once both the frontend and backend are running, sign up using dummy data and explore the available features.
